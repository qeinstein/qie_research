\section{Reproducibility}
\label{app:repro}

All experiments use five fixed seeds $\{7, 42, 99, 1337, 2026\}$.
Dry Bean and high-dimensional parity were additionally run on seeds
$\{100, 200, 300, 400, 500\}$ to resolve underpowered comparisons identified
in the initial analysis.
Code, configs, and per-seed JSON result files are available at
\url{\TODO{repository URL}}.
Dataset preparation scripts are provided for all six externally sourced
datasets.
The runner produces deterministic output for any given seed on the same
hardware; minor floating-point differences may arise across platforms.
