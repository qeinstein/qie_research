# Source Directory

`src/qie_research/` is the importable Python package for all scientific software in this repository.

- `encodings/` contains quantum-inspired feature maps.
- `baselines/` contains classical comparators.
- `analysis/` contains evaluation, NTK, representational, and statistical tooling.
- `pipelines/` contains end-to-end execution steps.
- `utils/` contains shared helpers.

Exploratory logic that becomes important to the paper should graduate into this package.
